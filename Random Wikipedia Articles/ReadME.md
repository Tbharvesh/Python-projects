# Build a Random Wikipedia Article fetcher application with python
## Objective :
Create a basic python Random Wikipedia Article fetcher application which fetches a random article from wikipedia and shows the link to the user. you can go one step further and use webscrapping to scrap the article and render it on your screen too.
<br>
Also add one of the following features:
- Make it user friendly by displaying the application somewhere rather than the terminal.
- Try to deploy it as a package onto pypi.org
